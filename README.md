# ADINN Roadshow - Next.js Version

This ZIP has been converted from the Lovable TanStack/Vite scaffold into a Next.js App Router project.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS v4
- Framer Motion
- React Three Fiber / Three.js
- Lenis smooth scrolling
- Outfit font via `next/font/google`

## Run

```bash
npm install
npm run dev
```

The dev server should show `next dev`. By default Next.js opens on:

```bash
http://localhost:3000
```

## Main files

- `src/app/layout.tsx` - Next root layout and metadata
- `src/app/page.tsx` - single page ADINN Roadshow site
- `src/styles.css` - Tailwind v4 theme and design system
- `src/components/site/` - page sections
- `public/assets/` - vehicle and CTA images
